# UltraEdit Website Layout Clone

A high-fidelity front-end reconstruction of the UltraEdit homepage, built to demonstrate proficiency in modern CSS layout techniques, including Flexbox and CSS Grid.

## 🚀 Live Demo
[View the live project here](https://arpitverma.github.io/ultraedit-clone/) *(Update this with your actual GitHub Pages link)*

## 🛠️ Technical Highlights

* **Semantic HTML5:** Used structured elements like `nav`, `div` containers, and descriptive class names for better code readability.
* **CSS Grid:** Implemented for the "Editor Features" section to create a clean, organized 3x3 grid layout.
* **Flexbox Mastery:** Extensively used Flexbox for the navigation bars, hero alignment, and the complex multi-column footer.
* **Interactive Elements:** Added hover states and transitions on buttons and inputs for a better user experience.
* **Typography:** Integrated Google Fonts (Inter) to match the professional aesthetic of the original site.

## 📋 Project Structure

* **Navigation:** Features a promotional top-bar and a secondary brand-focused navigation menu.
* **Hero Section:** Focuses on the Windows download call-to-action with centered typography.
* **Features Grid:** A responsive display of software capabilities using custom icons.
* **Professional Footer:** A dark-themed, 4-column footer layout including a newsletter subscription form.

## 📖 Key Takeaways

While building this project, I focused on:
1.  **Alignment & Spacing:** Perfecting the `justify-content` and `align-items` properties to maintain visual balance.
2.  **Component Reusability:** Using consistent classes like `.l` for feature boxes to keep the CSS DRY (Don't Repeat Yourself).
3.  **UI/UX Details:** Implementing a stylized email subscription bar with integrated buttons.
